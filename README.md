## Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/devpolatto/openDoor-Reactron.git
```

Download the dependencies and then run the expo

```bash
# Install the dependencies
$ yarn install

# Start the project
$ yarn electron
```
